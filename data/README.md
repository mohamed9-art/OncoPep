# PepOnco data

This folder contains processed and derived data used in the PepOnco manuscript.

Raw peptide records were compiled from publicly available peptide resources, including dbAMPseq, APD3, CancerPPD2, and DCP, as described in the Materials and Methods. Raw database files are not redistributed here unless permitted by the original source terms.

The `processed/` folder contains the standardized PepOnco corpus used for downstream analysis.
The `splits/`, `conditioning/`, and `tokenization/` folders are reserved for split assignments, train-derived condition schemas, and tokenization metadata.
