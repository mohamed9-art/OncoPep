# Benchmarking outputs

This folder is reserved for benchmark summaries comparing PepOnco with retrieval, conditional GRU, random conditional, unconditional GRU, and unconditional VAE baselines.

Recommended files include validity, uniqueness, exact novelty, condition fidelity, nearest-neighbour similarity, and per-condition recovery summaries.
