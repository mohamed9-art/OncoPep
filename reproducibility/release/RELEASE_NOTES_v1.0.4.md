# OncoPep v1.0.4

This release synchronizes the OncoPep reproducibility package with the
revised PLOS Computational Biology manuscript.

## Main changes

- Removed the former generator-benchmarking main Figure 2.
- Retained the generator benchmark and its numerical evidence under
  Results 3.5.
- Renumbered the former main Figures 3–5 as Figures 2–4.
- Renumbered the corresponding main-figure source-data files.
- Renamed the active Figure 4 and Figure 5 generation scripts to match
  the revised Figure 3 and Figure 4 numbering.
- Updated README and benchmarking documentation.
- Preserved historical v1.0.2 and v1.0.3 release-integrity records.

## Current main figures

1. Source composition and descriptor heterogeneity.
2. Reference-supported descriptor profiles of prioritized candidates.
3. Multi-objective prioritization of generated candidates.
4. Contextual sequence similarity and residue composition.

## Benchmark preservation

The deleted figure's underlying generator-benchmark data remain available at:

`results/benchmarking/source_data/Results_3_5_generator_benchmark_source_data.csv`

No reported benchmark result was removed or changed.
